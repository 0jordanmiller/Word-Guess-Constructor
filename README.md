# Hangman-with-constructors

A CLI Hangman application using Node.js, constructors and the npm package Inquirer 

## Getting Started
```
git clone git@github.com:0jordanmiller/Word-Guess-Constructor.git
```

Run ```npm i``` then ```node index.js```

## Built With

* Node.js
* Inquirer
